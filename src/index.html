Public-key cryptography made easy.

Public key cryptography is an essential technology to communicate secrets online, but can be confusing to understand and hard to use. Here we provide you with a tool that makes it easy to:

  1. Encourage others to use public-key cryptography when sending you files meant for your eyes only.
  2. Encrypt files with content you prefer to be kept secret.
  3. Decrypt the files to restore their original content.

Public-key cryptography is like having a padlock with one key code to lock it and another secret key to unlock. That way, anyone can send you a secret by locking it, while only you get access.
